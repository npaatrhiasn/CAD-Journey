# CAD-Journey

This repository documents my CAD skill development journey across two major platforms: PTC Creo and Dassault Systèmes CATIA. Each portfolio showcases both the validation of fundamental CAD competencies and their application in a complete "masterpiece" project.

## Repository Contents

| File | Description |
|------|-------------|
| `Portefolio Creo.pptx` | Creo portfolio containing two parts: <br>1. **CAD skills validation** – exercises and assessments demonstrating proficiency in Creo (modeling, assemblies, drawings, etc.). <br>2. **Masterpiece project: Steam engine** – Complete design of a steam engine, applying all validated skills to a functional mechanical system. |
| `Portefolio Catia.pptx` | CATIA portfolio containing two parts: <br>1. **CAD skills validation** – exercises and assessments demonstrating proficiency in CATIA (part design, assembly, surface modeling, etc.). <br>2. **Masterpiece project: Robotic arm** – Complete design of a robotic arm, showcasing advanced CAD capabilities and system integration. |

## Project Overview

### Creo – Steam Engine
- Design of a fully functional steam engine model

### CATIA – Robotic Arm
- Design of a multi-axis robotic arm

## How to Use This Repository

Download the PowerPoint files to view:
- Step-by-step skill validation pieces
- Screenshots of 3D models and assemblies
- Explanations of design choices and workflows
- Final rendered images and technical drawings of the masterpiece projects

The files are self-contained and can be opened with Microsoft PowerPoint or compatible viewers.

## Skills Demonstrated

- **Part modeling** (primitives, sweeps, lofts, holes, fillets, patterns)
- **Assembly design** (constraints, degrees of freedom, interference checking)
- **Engineering drawings** (dimensioning, tolerances, annotations)
- **Surface modeling** (CATIA specific)
- **Parametric design and design intent**
- **Motion simulation basics**
